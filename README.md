
# Chat Application

Free Chat Apps for Messaging on Your Phone or Computer


## Installation

 - [Clone Project](https://github.com/jithenderpara/ChatApplication/tree/develop)
 


## Run The Application

To run this project

```bash
  npm run install
  npm start watch
```
To lunch application in local
- [local](http://localhost:3000/?name=jithu)
